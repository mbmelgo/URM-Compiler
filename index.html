Unlimited Register Machine(URM) Compiler

Syntax

succesor (A); // increments the value of index A by 1
decrement (A); // decrement the value of index A by 1
zero (A); // makes the value of index A be equivalent to 0
copy (A,B); // makes the value of index B be equivalent to the value of index A
jump (A,B,C); // jumps to line C if the value of index A is not equivalent to the value of B, else proceed to the next line
print (A); // prints to the console the current value of index A
print ("STRING"); // prints to the console the STRING parameter

Keyboard Shortcuts

F7 - Key to Open or Close Console and Logger Interface
F8 - Key to Open or Close Info Interface
